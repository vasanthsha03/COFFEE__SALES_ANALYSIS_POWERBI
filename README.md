## 📊 Overview

This project showcases an interactive Coffee Shop Sales Dashboard built using Power BI. It provides insights into sales performance, customer behavior, product trends, and time-based analysis to support data-driven decision-making.

## 🚀 Key Features
### 🔹 Sales Performance
Total Sales: $699K
Total Quantity Sold: 214K
Total Transactions: 149K
Month-over-month growth indicators included for all KPIs
### 🔹 Sales by Location
Geographic visualization of sales across regions
Helps identify high-performing areas
### 🔹 Top Products Analysis
Top 10 best-selling products based on:
Total Sales
Number of Transactions
Popular items include:
Barista Espresso
Brewed Chai Tea
Hot Chocolate
### 🔹 Time-Based Insights
Sales segmented into:
Morning
Afternoon
Evening
Helps understand peak revenue periods
### 🔹 Peak Hours Analysis
Identifies top 3 hours with highest transaction volume
### 🔹 Product Performance Trends
Monthly sales trends for top 5 products
Tracks growth and seasonal patterns
### 🔹 Transaction-Based Product Insights
Top product details analyzed across:
Morning
Afternoon
Evening
### 🎛️ Filters & Interactivity
Month Selector (Jan–Jun)
Day Filter (Monday–Sunday)
Category Filter
Location Filter

These slicers allow dynamic exploration of the dashboard.

## 🛠️ Tools & Technologies Used
Power BI – Data visualization & dashboard creation
Excel / Dataset – Data source
DAX – Measures and calculations
Power Query – Data transformation

## 📈 Business Insights
Morning time generates the highest revenue
Certain beverages consistently outperform others
Sales show steady growth across months
Peak transaction hours help optimize staffing and operations
